서비스

① 데이터/지표 구조 진단

현황 데이터(엑셀/DB/툴) 기준으로 지표 정의, 누락/중복, 보고 체계를 점검합니다.

  • 지표(정의서) · 대시보드 목적 정렬
  • 데이터 소스/권한/수집 방식 점검

② SQL 기반 추출·집계

분석에 필요한 테이블 설계와 쿼리 작성, 자동 집계 구조를 구축합니다.

  • 데이터 정리(키, 조인, 집계)
  • 정기 리포팅용 뷰/쿼리 설계

③ BI 대시보드 구축/개선

Tableau 등 BI 도구로 KPI 모니터링, 원인 분석, 캠페인/매출 리포트를 구현합니다.

  • 스토리보드/레이아웃 설계
  • 운영을 고려한 필터/드릴다운

진행 방식

프로젝트 단계

  1. Kick-off: 목표/지표/사용자 정의
  2. 데이터 점검: 소스·컬럼·품질·키 구조
  3. 모델/쿼리: SQL 집계·뷰·데이터 마트
  4. 대시보드: KPI/분석 화면 구현
  5. 정착: 운영 가이드·교육·개선 루프

* 범위에 따라 1~4주 MVP 또는 단계별 확장으로 진행합니다.

산출물 예시

  • KPI 정의서(지표 산식/기준/담당)
  • 데이터 구조/ERD 또는 스타 스키마(필요시)
  • SQL 쿼리/뷰/집계 로직 문서
  • BI 대시보드(운영용/보고용)
  • 운영 가이드(업데이트/품질 점검 체크리스트)

주요 기술

“기술”은 화려함보다 의사결정에 바로 쓰이는 실무 구현력에 집중합니다.

Excel 분석·자동화

정제/피벗/함수/템플릿으로 초기 운영 구조를 빠르게 만듭니다.

SQL 추출·집계

JOIN/집계/뷰 설계로 반복 리포팅을 자동화합니다.

BI 대시보드

Tableau 등으로 KPI 모니터링/원인 분석 화면을 구현합니다.

기술 스택(예): Excel · SQL · BigQuery · Tableau · Power BI · GA4(옵션)
